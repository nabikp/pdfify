# PDFify

PDFify is a frontend utility that converts text or Markdown files into PDFs with preview.

## Features
- Markdown support (# headings, **bold**, *italic*, lists)
- Drag & drop `.txt` / `.md` file upload
- PDF preview before download
- Client-side only (no backend)

## Tech Stack
- React
- Vite
- jsPDF

## Run locally

```bash
npm install
npm run dev
